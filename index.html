JonahJS

The only JS library you'll ever need.

JonahJS is an amazing, lightweight JavaScript library that expands your JavaScript capabilities like no other. JonahJS takes your sloppy, badly-written JavaScript and recompiles it into attractive, sexy and loveable JavaScript that you just need to pet.

With its less than 1kb minified & gzipped size, JonahJS outplays literally every other JS Library in existence. It's also completely semantic, so you code kiddies that need a daily semantic fix can eat your heart out.

What can JonahJS do?

  • Recompile sloppy JS into beautiful, semantic JS.
  • Interpret CoffeeScript™ & TypeScript™ in realtime while your website is deployed. No need for those pesky Gulp/Grunt setups to compile your code!
  • Create CSS stylesheets with ReactJS™© intergration, so you can be the cool kid on the block.
